# Advent of Code

## Requirements

### Talking with [AoC](https://adventofcode.com) Website

- bash
- [httpie](https://github.com/httpie/cli)
- [Gum](https://github.com/charmbracelet/gum)

## Usage

### Set Cookie

```sh
./run.sh -c
```

### Get Input

Download the input for the current year and day to YEAR/DAY/input.txt

```sh
./run.sh -y 2023 -d 1
```
